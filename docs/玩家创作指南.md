# 陪你一起冒险 · 世界创作指南

> 这份文档告诉你：**怎么设计一个你自己的文字冒险世界**。

---

## 概述

一个「世界卡」包含六样东西：

| 模块 | 是什么 | 例子 |
|------|--------|------|
| **世界观** | 一段描述，告诉 AI 这是什么世界 | 蒸汽朋克天空城、东方玄幻王朝、赛博朋克东京 |
| **玩家属性** | 这个世界里「重要的能力」是什么 | 勇气、灵力、科技、声望……你自己定 |
| **NPC** | 关键人物，有好感度 | 铁匠老王（初始好感 20）、女巫艾琳（好感 10） |
| **物品** | 玩家开局带的 + 剧情中获得的道具 | 生锈的钥匙、加密信件、铜币 x3 |
| **旗标** | 不可逆的剧情标记，记录「发生了什么」 | 找到线索、结盟成功、真相揭露 |
| **节拍链** | 故事的大纲：先发生什么 → 再发生什么 → 终点 | 苏醒 → 找线索 → 调查矿坑 → 揭示真相 |

AI 读取这六样东西，在框架内自由发挥。你定义边界，AI 填充细节。

---

## 一、玩家属性：怎么影响剧情

### 属性是什么

属性是玩家的「能力值」，范围 0-10。你定义这个世界里什么是重要的。

```
蒸汽苍穹的例子：
⚔️ 勇气 (初始 3)  — 影响说服、对抗、冒险选项
🧠 智力 (初始 5)  — 影响解谜、分析线索、发现隐藏信息
💬 魅力 (初始 3)  — 影响社交、谈判、获取NPC信任
🔧 机械 (初始 4)  — 影响修理、操作机械、破解机关
```

### 怎么影响剧情

属性的影响通过**三个渠道**起作用：

**渠道 1：解锁选项。** 在节拍或选项上设置门槛。

```
例子：与贵族对质 这个节拍要求 courage >= 5。
勇气不够 → 选项不出现 → 你不能直接对质 → 得先通过别的途径攒勇气。

又如：操作飞艇残骸的通讯器 要求 mechanical >= 4。
机械不够 → 操作失败 → AI 生成失败场景 → 你获得不同的体验。
```

**渠道 2：影响 AI 叙事。** AI 会根据你的属性调整叙述。

```
勇气 8 → AI 描述：「你毫不畏惧地推开大门，守卫被你的气势镇住。」
勇气 2 → AI 描述：「你手抖得厉害，推了三次才把门推开一条缝。」
```

**渠道 3：属性可以在剧情中增减。** AI 根据你的行为判断属性变化。

```
你冒险爬上了坠落飞艇的残骸 → courage +2
你仔细研究了核心水晶的构造 → intellect +1
你对 NPC 撒了谎 → charm -1
```

### 设计建议

- **3-5 个属性**刚好。太少没区分度，太多记不住。
- **每个属性有明确的「适用场景」**。别设一个从来用不上的属性。
- **初始值不要太高**（3-5 合适），留增长空间。
- **属性门控的意义不是卡人，是指引**——告诉玩家「提升这个能力会有新选项」。
- **给不同属性设计不同的路线**。高勇气的玩家走正面冲突，高智力的玩家走破解谜题，两种路线都能推进故事，但体验不同。

## 二、NPC：怎么起作用

### NPC 是什么

世界里的关键角色，有好感度（0-100）。好感度由 AI 根据玩家行为调整。

```
例子：
铁鹰城 NPC：
├── 老机械师陈 (好感 20) — 对核心水晶了如指掌，但脾气古怪
├── 卫队长赵 (好感 0) — 忠于职守，对贵族不满但不敢对抗
└── 天空贵族洛 (好感 10) — 对机械着迷的异类贵族
```

### NPC 怎么影响剧情

**渠道 1：好感门槛。** 好感不够，NPC 不理你。

```
揭示真相 节拍要求 old_mechanic >= 40。
好感 20 → 老机械师对你态度冷淡，不愿透露更多。
你需要帮他修好机械臂（好感 +15）、带酒去看他（好感 +10），
好感够了，他才告诉你核心水晶的秘密。
```

**渠道 2：AI 根据好感调整 NPC 行为。**

```
好感 60 → 「老陈远远看见你就咧嘴笑了，连忙把你拉进工坊。」
好感 10 → 「老陈抬头看了一眼是你，继续低头摆弄零件，哼了一声。」
好感 -10 → 「老陈看见你就开始骂人了。」
```

**渠道 3：低好感 NPC 可能变成障碍。**

```
卫队长赵好感 -20 → 他在城门设卡，不让你通过。
你需要绕路、说服其他人担保、或者做任务补好感。
```

**渠道 4：不同 NPC 的好感互相影响。**

```
你帮了天空贵族洛 → 她喜欢你（好感 +10）
→ 但她父亲讨厌有人接近女儿 → 贵族阵营对你警惕
→ 卫队长赵反而因此对你态度好转（你们有了共同敌人）
AI 可以自然地处理这种连锁反应。
```

### 设计建议

- **写清楚 NPC 是谁、想要什么、怕什么**。AI 看到「他妻子三年前失踪」会自动生成相关剧情。
- **初始好感不是随机的**，暗示关系起点（0 = 陌生人，20 = 熟人，-10 = 有旧怨）。
- **给每个 NPC 一个剧情贡献点**——他在故事的什么环节起作用？
- **NPC 至少 2-3 个**，太少没互动，太多 AI 记不住。

## 三、物品：怎么用

### 物品是什么

字符串标签，没有数值。AI 根据世界观判断物品的意义。

```
例子：
开局物品: 机械扳手、残破的飞艇日志
剧情获得: 核心水晶碎片、贵族密信、工坊钥匙
剧情消耗: 用钥匙打开门（消耗）、把密信交给 NPC（消耗）
```

### 物品怎么影响剧情

**渠道 1：解锁选项。** 有物品才能做的事情。

```
调查矿坑 节拍要求 持有「机械扳手」。
你有 → 可以调查 → 发现矿坑里的线索。
你没有 → 需要先找到扳手（从老机械师那里借？从废墟里挖？）。
```

**渠道 2：AI 看到你持有物品，会围绕它生成剧情。**

```
你持有「残破的飞艇日志」→ NPC 看到会说「那是我参建的最后一批飞艇…」
你持有「铜币 x3」→ 可以在茶馆买情报，贿赂守卫，或只是买碗面吃
```

**渠道 3：物品可以被消耗、交换、丢失。**

```
你给了 NPC 一封信 → itemsLost: ["加密信件"]，NPC 好感可能变化。
你修好了什么 → itemsGained: ["修复的通讯器"]。
```

### 设计建议

- **开局物品暗示了世界的基调**。不要给无意义的物品。
- **给关键物品设计一个用途**——它在什么节拍或场景里会用到？
- **物品可以触发 NPC 反应**。你拿着他的东西出现在他面前……
- **不需要太多**。2-3 件开局物品 + 剧情中途获得，就够了。

## 四、旗标：记录「发生了什么」

### 旗标是什么

不可逆的布尔标记。`true` 表示「发生过」，`false` 表示「没发生过」。AI 自动管理。

```
例子：
found_crystal_clue (true) — 你找到了水晶的线索
allied_with_mechanics (false) — 还没和机械师公会同流
confronted_nobles (false) — 还没正面接触贵族
discovered_truth (false) — 还没发现真相
```

### 怎么用

**主要用途 1：控制节拍解锁。**

```
discover_truth (揭示真相) 需要 confronted_nobles 和 found_mine_clue 都完成。
完成 → 自动解锁 → AI 引导你进入真相揭晓的场景。
```

**主要用途 2：控制选项可见性。**

```
你之前救了冰面上的人 → saved_worker = true
→ 矿工们认出你来，愿意带你走密道（新选项出现）。
如果你没救 → saved_worker = false
→ 矿工们不认识你，你得另想办法。
```

**主要用途 3：AI 叙事参考。**

```
confronted_nobles = true → AI 知道「玩家已经和贵族撕破脸了」。
→ 贵族区域的 NPC 对你的态度会改变。
→ 新的风险出现（贵族派人跟踪你）。
```

### 设计建议

- **旗标是故事的记忆**。你做了什么事，旗标就记录什么。
- **旗标名字要有意义**。`betrayed_king`、`found_clue`——一看就知道什么意思。
- **旗标会出现在「已解锁旗标」面板里**，玩家也知道自己做了什么。
- **旗标可以和物品、NPC 好感联动**。完成 `confronted_nobles` → 贵族好感 -20，但卫队长好感 +15。

## 五、节拍链：设计故事走向

### 节拍是什么

节拍是故事的大纲节点。你不需要写每句对话，只需要定义**谁在什么条件之后能做什么**。

```
蒸汽苍穹的节拍链：
intro 坠毁醒来
  ↓
find_clue 发现线索
  ↓
  ├── confront_noble 与贵族对质  (需要勇气 ≥ 5)
  └── investigate_mines 调查矿坑 (需要持有「机械扳手」且机械 ≥ 4)
  ↓
discover_truth 揭示真相 (需要老机械师好感 ≥ 40)
```

### 每个节拍包含什么

```
{
  节拍名: "发现线索"
  节拍描述: "找到关于核心水晶下落的第一个线索——来自坠毁现场或老机械师"
  前置条件: (无)  // 一个节拍的 unlocks 列表包含它，它就解锁
  效果: {  // 完成后发生什么
    newFlags: ["found_crystal_clue"]  // 标记完成
  }
  解锁: ["confront_noble", "investigate_mines"]  // 完成后哪些节拍出现
}
```

### AI 在节拍框架里怎么工作

1. AI 看到你当前在哪个阶段、哪些节拍可解锁、哪些还不能碰
2. 你自由行动——探索、对话、试验
3. 当你的行动**符合某个可用节拍的描述**时，AI 判定「完成」，自动加旗标
4. 旗标驱动下一批节拍解锁
5. 节拍之间的空白，AI 完全自由发挥

### 设计建议

- **第一个节拍永远是不需要前置条件的**（初始化场景）。
- **节拍不要太密也不要在稀**。5-8 个节拍是一条合理的链。
- **让节拍之间有选择感**。不是一条直线，而是两条平行的解锁。
- **终端节拍的 unlocks 留空**（故事到此结束，但不阻止 AI 继续叙事）。
- **节拍描述要写清楚「完成标准」**。AI 需要判断玩家是否完成了这个节拍。

## 六、创作检查清单

在发布你的世界卡之前，检查一遍：

- [ ] 世界观描述够不够详细？AI 能否从中理解这个世界的基调？
- [ ] 属性定义清楚了吗？每个属性在什么场景发挥作用？
- [ ] NPC 有背景故事吗？AI 能根据背景生成合理的对话吗？
- [ ] NPC 的初始好感合理吗？（0=陌生人，20=熟人，负数=有矛盾）
- [ ] 开局物品有用途吗？它们在哪些节拍中会被用到？
- [ ] 旗标命名有意义吗？玩家看到能理解吗？
- [ ] 节拍链是否合理？从前到后，条件是否越来越难？
- [ ] 有没有给不同属性的玩家设计不同的路线？
- [ ] 终端节点有明确的完成感吗？

## 附录：完整示例

### 蒸汽苍穹（steampunk_skyfall）完整规格

**世界观**：蒸汽朋克时代，魔法与机械并存。七座天空之城靠核心水晶悬浮。铁鹰城的核心水晶被窃，城市正在坠落。你的身份：一名年轻的机械师学徒。

**属性**：勇气(3)、智力(5)、魅力(3)、机械(4) —— 各 0-10

**NPC**：
- 老机械师陈（好感 20）：退休首席机械师，妻子三年前失踪，对水晶了如指掌
- 卫队长赵（好感 0）：忠于职守，内心深处对贵族不满
- 天空贵族洛（好感 10）：年轻女贵族，对机械充满好奇，渴望冒险

**开局物品**：机械扳手、残破的飞艇日志

**旗标**：found_crystal_clue、allied_with_mechanics、confronted_nobles、discovered_truth

**节拍链**：
```
坠毁醒来 → 发现线索 → [与贵族对质（需勇气5）] + [调查矿坑（需机械4+扳手）]
                                             ↘           ↙
                                           揭示真相（需老机械师好感40）
```
